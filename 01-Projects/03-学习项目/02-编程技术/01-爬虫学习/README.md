# 01-爬虫学习项目

## 项目说明
此文件夹用于存放爬虫学习相关的代码和笔记，仅供个人学习使用。

## 目录结构
- `code/`: 存放爬虫代码
- `notes/`: 存放学习笔记
- `data/`: 存放爬取的数据（如有需要）
- `resources/`: 存放学习资源和参考材料

## 学习目标
1. 掌握基本的网络爬虫原理和技术
2. 学习常用爬虫库的使用（如Requests、BeautifulSoup、Scrapy等）
3. 理解网页结构和数据提取方法
4. 熟悉网站反爬虫机制及应对策略
5. 实践数据存储和分析

## 注意事项
- 本项目仅用于学习目的，请遵守相关法律法规
- 爬取网站数据时，请注意控制请求频率，避免对目标网站造成负担
- 对于有robots.txt限制的网站，请予以尊重
- 不得将爬取的数据用于商业用途
