# 刘老师词汇课精选

## Criteria / Criterion (批判标准)
  #1
-   **发音助记**: "批判" -> 批判一个人需要有标准 (criteria / criterion)
-   **词义**: 标准, 准则
-   **派生词**:
    -   `criterion` (n. sing.): 标准, 准则 (单数形式)
    -   `critical` (adj.): 批判的, 关键的
    -   `critic` (n.): 批评家, 评论家

## Irony (铁的讽刺)
  #1
-   **发音助记**: `Iron` (铁) -> 说话像铁一样冷冰冰 -> 讽刺
-   **词义**: (n.) 反语, 讽刺, 具有讽刺意味的事
-   **派生词**:
    -   `ironic` (adj.): 讽刺的, 用反语的
