# 第一章 统计学导论：变量与数据

## 章节框架概述

本章首先介绍统计分析的原料——**变量和数据**，随后探讨统计学中的**基本概念**（这些概念蕴含了统计学的基本思想），最后阐述**统计学的定义及其应用领域**。这与传统教材先定义统计学再展开其他内容的顺序略有不同。

## 一、变量与数据 (Variables and Data)

这部分我们将探讨变量和数据的概念及其类型。

### 1. 变量 (Variable)

**定义**：变量是用来描述现象某种特征的概念。

为了理解这个定义，我们先拆解：

* **现象 (Phenomenon)**：指我们能够感知到的一切事物或情况。
    * 例如：刮风下雨（自然现象），人们准备过年（社会现象）。
* **特征 (Characteristic)**：现象从不同角度所表现出的特性。
    * 以“风”为例，它是一种现象。其特征有：力量大小、方向等。
* **概念 (Concept) / 变量 (Variable)**：为说明现象的这些特征而产生的相应概念。
    * 用“风力”描述风的力量大小。
    * 用“风向”描述风吹来的方向。
    * “风力”和“风向”就是变量。
* **变量的特点**：在于其“变”。不同时间、不同地点，变量的取值可能不同。
    * 例如：天气预报中，有的地方东风三级，有的地方南风五级。“三级”和“五级”是风力这个变量的不同取值。

### 2. 数据 (Data)

**定义**：变量的取值称为**变量值 (Variable Value)**，又称为**数据 (Data)**。

* 简单来说，数据可以视为**对现象进行测量的结果**。
* 从现象到数据，需要一个**量化 (Quantification)**的过程，即**测量 (Measurement)**。

### 3. 数据的常用组织形式

数据通常以表格形式组织，包含以下要素：

| 观测对象 (Observation Unit) | 变量1 (Variable 1) | 变量2 (Variable 2) | ... |
| :-------------------------- | :----------------- | :----------------- | :-- |
| 张三                        | 男                 | 5000               | ... |
| 李四                        | 女                 | 8000               | ... |
| ...                         | ...                | ...                | ... |

* **观测对象**：表格的第一列通常是观测对象（也称观察单位、个体）。
* **变量**：如“性别”、“月收入水平”是变量。
* **数据**：如“男”、“女”、“5000”、“8000”是相应变量的取值，即数据。
* **数据的归属与含义**：有了观测对象和变量，数据就有了明确的归属和含义。例如，“男”这个数据指的是“张三的性别是男”。
* **省略观测对象**：有时表格可能没有明确列出观测对象，此时默认同一行中的数据来源于同一个观测对象。

## 二、数据的类型 (Types of Data)

数据按照不同的分类标准有不同类型，主要有三种分类标准，其中**按计量尺度分类**最为重要。

### 1. 按计量尺度 (Measurement Scale) 分类

从现象到数据需要通过“测量”，不同的测量方法和标准对应不同的**计量尺度**。计量尺度不仅种类不同，其层次水平也有高低之分。我们从低到高介绍四种计量尺度：

#### a. 分类尺度 (Nominal Scale)

* **等级**：最低的计量尺度。
* **测量结果**：类别 (Categories)。
* **特点**：类别之间是平行的，没有顺序或等级之分。通常用文字表示。
* **示例**：
    * 性别：男、女。尺子的刻度可以表示为【男】和【女】，顺序可互换。
    * 婚姻状态：未婚、已婚、离婚、丧偶。
* **数值编码**：有时为了统计方便，会用数字作为类别的标签或编码。
    * 例如：用 $$1$$ 代表“男”，$$0$$ 代表“女”。这里的 $$1$$ 和 $$0$$ 仅代表类别不同，不具有数学大小意义。此时通常会有额外说明，如“$$1$$ 表示男，$$0$$ 表示女”。
* **数学特征**：仅有“相等”($$ = $$) 和“不相等”($$ \neq $$)。
    * 两个人的性别，要么相等，要么不相等。

#### b. 顺序尺度 (Ordinal Scale)

* **等级**：比分类尺度高一个等级。
* **测量结果**：有序的类别 (Ordered Categories)。
* **特点**：类别之间不仅有区分，还有顺序或等级之分。
* **示例**：
    * 学历：文盲、小学、初中、高中、大学及以上。这些刻度表示了类别，也表示了从低到高的顺序（或从高到低）。
* **数据也称**：有序分类数据。
* **数学特征**：
    * 相等 ($$ = $$) 或不相等 ($$ \neq $$)
    * 大于 ($$ > $$) 或小于 ($$ < $$)
    * 例如：学历“小学”低于“大学”。
* **局限性**：
    * **无法测量差距**：不同类别之间的差距无法用减法精确测量。例如，不能说“小学 - 文盲 = 某个值”。
    * **无法测量比率/倍数**：不能进行乘法或除法运算。例如，不能说“小学 / 文盲 = 某个比值”。

#### c. 间隔尺度 (Interval Scale)

* **等级**：比顺序尺度更高。
* **测量结果**：数值 (Numerical Value)。
* **特点**：
    * 类别有顺序。
    * 类别之间的**差距是可测量且有意义的**。
    * **没有绝对零点 (Absolute Zero)**，其零点是人为设定的。这意味着当数据取值为 $$0$$ 时，被测特征并未消失。
* **示例**：
    * **温度 (摄氏度 $$ ^\circ C $$, 华氏度 $$ ^\circ F $$)**：$$0^\circ C$$ 并不表示没有温度，只是一个较低的温度水平，温度还可以更低（如负值）。
    * **智商 (IQ) 分数**：通过题目设计和分数分配来测量智力水平。若某人IQ测试得 $$0$$ 分，表示其智力水平在此次测试中表现较低，不代表其完全没有智力。零点是在人的智力水平变化范围内人为设定的。
    * **考试分数**：测量某种能力或知识掌握程度。得 $$0$$ 分表示在该次考试中表现为零，不代表完全没有该能力。
    * **性格测试分数**：例如，测试内向外向程度，$$0$$ 分可能代表极度内向，但不代表没有性格。
    * **日期**：如公元纪年，公元 $$0$$ 年（或 $$1$$ 年）是人为设定的参照点（通常关联耶稣诞生），此前仍有漫长历史。
* **数学特征**：
    * 相等 ($$ = $$) 或不相等 ($$ \neq $$)
    * 大于 ($$ > $$) 或小于 ($$ < $$)
    * 加法 ($$ + $$) 或减法 ($$ - $$) 来计算差值。
        * 例如：$$30^\circ C - 20^\circ C = 10^\circ C$$ 是有意义的。日期间的差值（时间段长度）也是可以计算的。
        * 加法运算的存在意味着可以计算**算术平均数 (Arithmetic Mean)**。例如，考试平均分：
            $$ \text{平均分} = \frac{\text{分数}_1 + \text{分数}_2 + \dots + \text{分数}_n}{n} = \frac{\sum_{i=1}^{n} \text{分数}_i}{n} $$
            其中：
            * $$ \sum $$ (Sigma)：表示求和。
            * $$ i=1 $$ 到 $$ n $$：表示对从第1个到第 $$ n $$ 个个体进行求和。
            * $$ \text{分数}_i $$：代表第 $$ i $$ 个个体的分数。
            * $$ n $$：代表个体总数。
* **局限性**：
    * **不能计算比率或倍数**：因为零点是人为的。说 $$40^\circ C$$ 是 $$20^\circ C$$ 的两倍是**不正确**的。

#### d. 比例尺度 (Ratio Scale)

* **等级**：最高的计量尺度。
* **测量结果**：数值 (Numerical Value)。
* **特点**：
    * 拥有间隔尺度的所有特性（有序、差值有意义）。
    * 拥有**绝对零点 (Absolute Zero)**。这意味着当数据取值为 $$0$$ 时，被测特征完全消失或缺乏。
    * 测量的起点是特征存在与不存在的界限。
* **示例**：
    * **重量/长度/身高**：$$0$$ 千克表示没有重量，$$0$$ 米表示没有长度。身高从地面（存在与不存在的界限）开始测量。
    * **收入**：收入为 $$0$$ 意味着没有收入。
    * **年龄**：年龄为 $$0$$ 意味着个体尚未出生（或刚出生瞬间）。
    * **人口密度**：$$0$$ 表示该区域没有人。
    * **练习时长**：$$0$$ 小时表示没有练习。
    * **温度 (开尔文 K)**：开尔文温标的零点 ($$0K$$) 是绝对零度，此时物质内部分子运动停止，温度特征消失。
* **数学特征**：
    * 相等 ($$ = $$) 或不相等 ($$ \neq $$)
    * 大于 ($$ > $$) 或小于 ($$ < $$)
    * 加法 ($$ + $$) 或减法 ($$ - $$)
    * 乘法 ($$ \times $$) 或除法 ($$ \div $$) 来计算比率或倍数。
        * 例如：$$100$$ 千克是 $$50$$ 千克的两倍 ($$ \frac{100 \text{kg}}{50 \text{kg}} = 2 $$) 是有意义的。
        * 乘法运算的存在意味着可以计算**几何平均数 (Geometric Mean)**。例如，对于三个值 $$x_1, x_2, x_3$$：
            $$ G = \sqrt[3]{x_1 \times x_2 \times x_3} $$
            其中：
            * $$ \sqrt[3]{\dots} $$：表示开立方根。
            * $$ x_1, x_2, x_3 $$：代表各个观测值。
            * 间隔尺度数据不能计算几何平均数，因为它涉及到乘法。

#### 解释：为什么 $$40^\circ C$$ 不是 $$20^\circ C$$ 的两倍？

* **摄氏度的零点**：摄氏度的 $$0^\circ C$$ 是人为设定的（水的冰点），并非绝对零点。
* **绝对零点与温度**：温度是物质内部分子运动速度的外在表现。当分子运动停止时，温度才真正为零，这是绝对零点（约等于 $$-273.15^\circ C$$ 或 $$0K$$）。
* **开尔文温标 (K)**：是以绝对零点为起点的比例尺度。
    $$ T_K = T_C + 273.15 $$
    其中：
    * $$ T_K $$：开尔文温度。
    * $$ T_C $$：摄氏温度。
    * $$ 273.15 $$：摄氏零度与绝对零度之间的差值（近似值，课件中用 $$273$$）。
* **正确比较**：
    * $$20^\circ C$$ 对应的开尔文温度约为 $$20 + 273.15 = 293.15 K$$。
    * $$40^\circ C$$ 对应的开尔文温度约为 $$40 + 273.15 = 313.15 K$$。
    * 它们之间的实际比率是：
        $$ \frac{313.15 K}{293.15 K} \approx 1.068 $$
        这个比值显然不等于 $$ \frac{40}{20} = 2 $$。
* **人为设定零点的影响**：
    如果测量身高时，人为规定从 $$1$$ 米处开始设为新的“零点”：
    * 某人身高 $$1.8$$ 米，新“身高”为 $$1.8 - 1 = 0.8$$ 米。
    * 蜡笔小新身高 $$1.2$$ 米，新“身高”为 $$1.2 - 1 = 0.2$$ 米。
    * 此时倍数关系为 $$ \frac{0.8}{0.2} = 4 $$ 倍。
    如果从 $$1.1$$ 米处设为新“零点”：
    * 某人新“身高”为 $$1.8 - 1.1 = 0.7$$ 米。
    * 小新新“身高”为 $$1.2 - 1.1 = 0.1$$ 米。
    * 此时倍数关系为 $$ \frac{0.7}{0.1} = 7 $$ 倍。
    可见，人为设定的零点会导致比率计算结果随设定方式改变，因此这种比率是无意义的。而真实的倍数（基于绝对零点，即从地面开始测量）是 $$ \frac{1.8}{1.2} = 1.5 $$ 倍，是固定的。
    **但是，差值不受影响**：
    * $$1.8 - 1.2 = 0.6$$ 米
    * $$0.8 - 0.2 = 0.6$$ 米
    * $$0.7 - 0.1 = 0.6$$ 米
    所以，绝对零点影响倍数计算，不影响差值计算。

#### 计量尺度层级总结

| 尺度类型     | 描述                                   | 数学特征                                       | 信息量/精确度 |
| :----------- | :------------------------------------- | :--------------------------------------------- | :------------ |
| **分类尺度** | 类别，无序                             | $$=, \neq$$                                    | 最低          |
| **顺序尺度** | 有序类别，差值无意义                   | $$=, \neq, >, <$$                              | 低            |
| **间隔尺度** | 数值，差值有意义，无绝对零点           | $$=, \neq, >, <, +, -$$ （可计算算术平均数）   | 中            |
| **比例尺度** | 数值，差值、比率均有意义，有绝对零点 | $$=, \neq, >, <, +, -, \times, \div$$ （可计算几何平均数） | 最高          |

* **信息量与精确度**：从分类尺度到比例尺度，数据携带的信息量越来越多，能展示的细节也越多，准确度或精确度也越来越高。
* **数学运算特征**：越来越丰富，直接影响后续数据分析方法的选择。

#### 将数据归纳为两大类：

1.  **分类数据 (Categorical Data) / 定性数据 (Qualitative Data) / 品质数据 (Attribute Data)**：
    * 由**分类尺度**和**顺序尺度**测量得到。
    * 反映事物品质或属性方面的特征。
2.  **数值数据 (Numerical Data) / 定量数据 (Quantitative Data) / 数量数据 (Quantity Data)**：
    * 由**间隔尺度**和**比例尺度**测量得到。
    * 表现形式为数值，反映事物数量方面的特征。

#### 数值数据的进一步划分 (仅针对数值型数据)：

数值数据根据取值不同，可分为离散型数据和连续型数据。**注意：分类数据没有离散和连续的区别。**

1.  **离散数据 (Discrete Data)**
    * **定义**：取值有限或可列举 (Countably Infinite)。
    * **有限取值**：如牙齿的数量、班级录取人数。通常表现为“...的数量”、“...的数目”、“...的次数”。
    * **可列举**：指在一个区间内只取整数，即使区间很大（如 $$[0, +\infty)$$），理论上只要有足够时间，可以从 $$0$$ 开始一个一个地数到任何一个目标值。例如，一年内某路口通过的汽车数量。
    * **判断依据**：
        * 通常是**计数 (Counting)** 得来的。
        * 其取值区间**不能无限细分**。若细分到一个不包含整数的区间，对该数据无意义。

2.  **连续数据 (Continuous Data)**
    * **定义**：取值无限且不可列举 (Uncountably Infinite)。其变化范围往往是一个或多个区间。
    * **示例**：温度（如 $$10^\circ C$$ 到 $$20^\circ C$$ 之间可以取任意值）、长度、重量。
    * **不可列举性**：在一个区间内，例如从 $$10$$ 开始，无法列出“下一个”紧邻的值，因为 $$10$$ 和任何 $$10+\epsilon$$ (epsilon是一个极小的正数) 之间都还有无穷多个值。
    * **判断依据**：
        * 通常是**测量 (Measuring)** 得来的。
        * 其取值区间**可以无限细分**。无论区间细分到多窄，总能取到该区间内的值。

### 2. 按收集方法 (Collection Method) 分类

这里强调的是数据的**初始来源**，即从现象到数据的过程是如何收集的。从统计局网站下载数据不在此讨论范围，因为那是数据已经产生后的转移。

#### a. 观测数据 (Observational Data)

* **核心**：“看看发生了什么？” (Look at what happened)
* **方法**：对已经发生的现象进行观察和记录，研究者**不施加人为控制或干预**。
* **示例**：社会经济数据（如GDP、CPI）大部分通过观测得到。

#### b. 实验数据 (Experimental Data)

* **核心**：“看看会发生什么？” (See what will happen)
* **方法**：研究者对现象的产生**施加人为控制**，主动改变某些条件（实验条件/变量），然后观察结果变量的变化。
* **示例**：研究病毒在不同材质、不同温度下的存活时间。
    * 控制变量：材质（不锈钢、木头）、温度（$$20^\circ C$$、$$30^\circ C$$）。
    * 观测变量：病毒存活时间。
    * 通过控制实验环境（材质和温度的组合），观察其对存活时间的影响。
* **常见领域**：自然科学数据大多通过实验得到。

### 3. 按现象与时间的关系分类

#### a. 截面数据 (Cross-sectional Data)

* **定义**：在**同一时间点（或时间段）**对**不同观测对象**的同一（组）变量进行观测得到的数据。
* **特点**：数据的变化维度体现在不同观测对象上，反映的是某个时间点的静态特征。
* **示例**：某一年年末，不同地区的常住人口数。

    | 地区 | 年末常住人口数 (某年) |
    | :--- | :-------------------- |
    | A    | 1000万                |
    | B    | 1500万                |
    | C    | 800万                 |

#### b. 时间序列数据 (Time Series Data)

* **定义**：对**同一观测对象**在**不同时间点（或时期）**上进行观测所得到的数据，按时间顺序排列。
* **特点**：数据的变化维度体现在时间上，反映某个观测对象的动态变化特征。
* **示例**：北京市从2000年到2020年每年的GDP。

    | 年份 | 北京市GDP |
    | :--- | :-------- |
    | 2000 | X1        |
    | 2001 | X2        |
    | ...  | ...       |
    | 2020 | Xn        |

#### c. 面板数据 (Panel Data) / 平行数据 (Parallel Data)

* **定义**：将截面数据和时间序列数据结合起来，是在**不同时间点**对**同一组不同观测对象**进行重复观测得到的数据。
* **特点**：变化维度是双重的，既可以在同一时间点上比较不同观测对象，也可以对同一观测对象比较不同时间点的情况。
* **示例**：多个地区在多个年份的年末常住人口数。

    | 地区/年份 | 2018年 | 2019年 | 2020年 |
    | :-------- | :----- | :----- | :----- |
    | 北京市    | X11    | X12    | X13    |
    | 上海市    | X21    | X22    | X23    |
    | 广州市    | X31    | X32    | X33    |

## 三、变量的类型 (Types of Variables)

变量的类型与数据的类型密切相关，因为变量的取值就是数据。

### 1. 按计量尺度分类

与数据按计量尺度分类完全对应：

* **分类变量 (Categorical Variable)**：其取值为分类数据（定类尺度或定序尺度得到的数据）。
* **数值变量 (Numerical Variable)**：其取值为数值数据（定距尺度或定比尺度得到的数据）。
    * **离散变量 (Discrete Variable)**：取值为离散型数值数据。
    * **连续变量 (Continuous Variable)**：取值为连续型数值数据。

### 2. 按随机性分类

* **随机变量 (Random Variable)**：其取值在观察前不确定，具有随机性。
    * 例如：掷骰子的点数，掷硬币的结果（正面/反面）。
* **非随机变量 (Non-random Variable / Deterministic Variable)**：其取值在特定条件下是确定的，虽然它仍然是“变量”（即在不同个体或条件下会变）。
    * 例如：在一个**特定**的班级中，考察同学们的性别。对于**特定**的张三，其性别是确定的（男）。但“性别”这个变量在班级这个群体中是变化的（有的男，有的女）。

### 3. 按经验与理论分类

* **经验变量 (Empirical Variable)**：在日常生活中可以被直接观察或感受到的变量。
    * 例如：人的高矮胖瘦、肤色、发型。
* **理论变量 (Theoretical Variable / Latent Variable / Construct)**：由统计学家为了统计分析或推断的需要而构造出来的一些量，在现实中不能直接观察到。
    * 例如：统计推断中的一些统计量，如卡方统计量 ($$\chi^2$$)、t 统计量 ($$t$$)、F 统计量 ($$F$$)。这些是为了特定统计检验而构造的理论概念。

本节详细介绍了变量与数据的概念、组织形式以及最重要的分类方式——特别是按计量尺度进行的分类，这对理解后续的统计分析方法至关重要。希望大家能对本节内容进行梳理和总结。
